#### Setup project:-


1. check node version

```````
	node --version
```````
2. Create React App using Vite

```````
	npm create vite@latest
```````
3. Move into directory

```````
	cd firstapp
```````
4. install node modules

```````
	npm install
```````
5. run the app

```````
	npm run dev
```````
